# LiFi-Prototype
Prototype to transfer ASCII encoded characters from transmitter to receiver via Visible Light Communication. 

### Components Required:
* 1x Solar Panal
* 3x LED
* 1x Resistance 330 Ohm
* 1x Arduino USB Cable
* 1x Arduino Uno R3
* Some wires and a breadboard
